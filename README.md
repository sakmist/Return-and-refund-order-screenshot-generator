# 退货退款订单截图生成器

[English](https://github.com/sakmist/Return-and-refund-order-screenshot-generator/blob/main/README_en.md)
这是一个用于生成退货退款订单截图的在线工具。它可以帮助用户快速生成看起来真实的退货订单截图，主要用于学习目的。

> ⚠️ 免责声明：此工具仅供学习交流使用，请勿用于伪造或误导他人。实际寄件信息必须与快递真实信息一致。

## 功能特点

- 快速生成拼多多退货退款订单截图
- 可自定义收货人姓名、电话和地址
- 支持从剪贴板自动解析并填充联系人信息
- 一键生成高清截图并下载
- 响应式设计，支持不同屏幕尺寸

## 使用方法

1. 打开 `return.html` 文件（推荐使用现代浏览器如 Chrome/Firefox）
2. 在右侧表单中填写相关信息：
   - 收货人（包括姓名和电话）
   - 收货地址
3. 点击「生成截图」按钮
4. 浏览器会自动下载生成的截图

### 自动填充功能

您可以复制一段包含姓名、电话和地址的文本，然后点击「粘贴自动填」按钮，系统会自动解析并填充相应字段。

示例格式：
```
收货人：张三 13812345678
收货地址：北京市朝阳区某某街道某某大厦123号
```

## 技术实现

- 使用 Vue.js 构建界面交互
- 使用 html2canvas 实现网页截图功能
- 配置化字段和图层显示
- 支持响应式布局

## 配置说明

项目通过 [config.json](config.json) 进行配置：

- `fields`: 定义表单字段
- `layers`: 定义截图中的图层元素及其样式

## 本地运行

只需在浏览器中打开 `return.html` 文件即可使用，无需额外服务器支持。
